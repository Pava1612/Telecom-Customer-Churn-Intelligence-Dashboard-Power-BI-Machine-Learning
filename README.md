# 📊 Telecom Customer Churn Intelligence Dashboard

## 📌 Project Overview

Customer churn is a major challenge for telecom companies. Losing customers directly impacts revenue and increases the cost of acquiring new customers.

This project analyzes telecom customer data to identify **churn patterns, customer segments, and revenue drivers** using **Python-based data analysis and an interactive Power BI dashboard**.

The goal is to provide **data-driven insights** that help telecom companies improve **customer retention strategies and business decision-making**.

---

## 🎯 Objectives

* Analyze telecom customer behavior and churn patterns
* Identify high-risk churn groups
* Segment customers using machine learning
* Build an interactive dashboard to visualize business insights

## 📂 Dataset

This project uses the **Telco Customer Churn dataset**.

The dataset contains telecom customer information including:

* Customer tenure
* Monthly charges
* Total charges
* Contract type
* Internet service
* Payment method
* Customer churn status

Total Records: **7032 customers**

## 💡 Key Business Insights

* Customers with **month-to-month contracts show the highest churn rate**
* Customers with **higher monthly charges churn more frequently**
* Customers with **tenure less than 12 months have the highest churn probability**
* **High-value customer segments contribute the majority of telecom revenue**

---

## 🧠 Machine Learning Component

Customer segmentation was performed using **K-Means clustering** based on:

* Tenure
* Monthly Charges
* Total Charges

This helped identify different **customer value groups** such as:

* Low-value customers
* Medium-value customers
* High-value customers
